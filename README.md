# Project

## Project info

This repository contains the source for the project built with Vite, React and Tailwind CSS.

## How can I edit this code?

You can edit the project locally using your preferred IDE. The only requirement is Node.js & npm.

Quick start:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Deploy using your preferred static hosting provider (Netlify, Vercel, GitHub Pages, etc.). Build the project with `npm run build` and follow your host's deployment steps.

## Custom domain

To connect a custom domain, follow your hosting provider's documentation for adding a domain to the site.
